Kills per Minute

This feature outputs a ratio of Kills divided by Minutes rounded to nearest hundredths place. The display is updated for each kill as well as every second. This allows higher accuracy than simply updating every minute.

The time starts as soon as the K/min window is opened and can be reset using the "Reset Counters" hotkey or via the button in Settings.

# Headshot

This feature simply displays the number of headshots achieved by the player. The counter starts when WarShadow launches and can be reset using the "Reset Counters" hotkey or via the button in Settings.

% Headshot

This feature displays the result of: <# of Headshots>/<# of Kills> X 100. It is updated after each kill. The counter starts when WarShadow launches and can be reset using the "Reset Counters" hotkey or via the button in Settings.

HS Streak Average

This feature shows how many headshots the player gets in quick succession on average.
Examples: If a player only got brain annhilators the window would always display "5 Avg". If a player never got a skull hunter, head crasher, brain smasher, or brain annhilator the window would always display "1 Avg".

The counter starts when WarShadow launches and can be reset using the "Reset Counters" hotkey or via the button in Settings.

Crosshair

This feature allows the user to use a custom crosshair in Warface. The user can use preloaded crosshairs or upload their own. Only shows up while in-game.

Setting it up

Open the Settings page and scroll down to the Crosshair section. If you use right click to aim down sights, make sure the box is checked. Select Hold-to-Aim or Toggle-Aim so that it is the same as your Warface settings. If you use Right Click to ADS you are done!
If you use a custom key to aim, click "Grab ADS" and push your ADS key. To make sure that the app recieved the correct key, push it again. If the text below the button flashes grey you are all finished!

"Uploading" your own crosshair

To use a custom crosshair all you need to do is create a folder called "Crosshair" and put it on your desktop. Then get a .png file of your crosshair and put it in the folder. See below for naming the file.

I have provided 10 "slots" for custom user crosshairs. The available file names are:
customX0.png, customX1.png, customX2.png, customX3.png, customX4.png, customX5.png, customX6.png, customX7.png, customX8.png, and customX9.png.

Disclaimer: Sometimes the settings page doesn't detect your ADS key and still displays "Waiting for ADS key". If this happens simply close and reopen Settings. I'm looking into the issue

General Recording Info

Q: Does the recording capability save past gameplay like ShadowPlay or does it do start-stop recording like more traditional programs?
A: Both. Auto-Recording works like ShadowPlay, but the recording window has the option to do start-stop recordings as well.

Q: Why won't the Auto-Recording box stay checked?
A: It is designed to uncheck itself if the recording process didn't start correctly. This is usually because it wasn't clicked when the user had a game running and in focus. If this wasn't the issue, read more in the Problems section below.

Q: Why does the program lag when I click Auto-Record or I click the power button in the recording window?
A: These buttons are both enabling the ShadowPlay feature and it takes a few seconds for Overwolf to get everything ready.

Q: Why didn't my recording save?
A: Do you remember the slider in Settings that lets you choose how much video you wanted to save after the auto-Recording event happened? When an Auto-Record event is triggered, my app waits that long before saving the video. The reason for this, is because Overwolf only allows one recording at a time. If I didn't wait till after the time was finished, other Auto-Recording events are unsuccessful because the first call is hogging the recording.

Q: Why should I use Warshadow for recording instead of ReplayHUD?
A: 3 reasons.
1) "Auto-Recording", it takes all the recording for you so you can focus on playing the game.
2) The option to do longer videos. ReplayHUD currently only allows 40 second videos. Mine does 3 minutes for Auto-Recording, but I haven't found a time limit for start-stop recording.
3) No recharge time. ReplayHUD currently makes you wait 40 seconds before taking another recording.

Auto-Recording

This is probably the most useful feature in my app. AutoRec records your best moments for you, so you can focus on the game. No more massive recordings that you comb through later, no more fumbling for the "Save video" Hotkey.

Simply select which moments you want to save (this is done in settings) and click on Auto-Record in the main menu. Whenever an event happens, the app will save the video recording for you and you never have to worry about it.

You can choose to change the length of the recording. Don't worry about making the times too short, the app has a built in time for each event so that even if you say you want a 2 second recording, you will still get video of your whole brain annihilator.

Why is the damage graph on Shotguns so weird?

I'm so glad you asked! Its because I made it up. Kinda. All of the shotguns in the game have a certain number of pellets. That is what makes shotguns great! The catch is that my calculations assumes that the user has 100% accuracy.

That becomes a problem because at long distances it isn't possible to shoot the enemy with ALL of the pellets (without hacks). Since there is no real way for me to predict the spread patterns I made up a formula:
Every 1 meter (roughly) the chance of hitting with all pellets reduces by 1%. Then I use that % of the guns pellets in the calculation.

Thats why the graph is all bumpy, at that distance 1 less pellet hit the enemy.

Ok so then what good are the shotgun calculations?
I had this question myself, and I decided that the chart is only good for comparing shotguns against other shotguns. Comparing against any other weapon is just a guess.
