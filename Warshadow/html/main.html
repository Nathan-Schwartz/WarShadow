Would you like to go over the basics?
To collapse the main menu, simply double click on the Warface Logo.
Auto-Record takes care of saving your gameplay, so you can just play the game.
If you have any problems with the recording feature, check the "Recording" tab on the info page.
In the Settings page you can select a different UI that better blends in with Warface.
Warshadow has some HUD features to track statistics. Open via checkboxes.
Don't forget to check out the Smoke Timer hotkey!
You can use custom crosshairs in the Crosshair feature and even add your own!
Click on the 'info' button to learn know more.

You cannot use AutoCapture and manual recording at the same time.
